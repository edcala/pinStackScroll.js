pinStackScroll.js

This plugin pins and stacks selected elements at the top of the screen as the elements are scrolled to the top of the viewport

    Usage:

  • Include pinStackScroll.js in the <head> section after including jQuery
  • Include pinStackScroll.css styling in the <head> section

  • Run this in the <body> after the document is loaded:

    <script>
	$(document).ready(function(){
		$(".stackElement").pinStackScroll();
	});
<script>
  • Done!

    Options:

  • overlapStack: (true||false)

    true - The static divs appear when the top of the original divs clips the top of the viewport.

    false - (default) The static divs appear when the top of the original divs clips the bottom of an existing static div.

  • hideOriginals: (true||false)

    true - Hide the original elements (make invisible) when a static element appears.

    false - (default) Keep the originals visible when a rstatic element appears.

  • remainOnTop: (true||false)

    true - Stacked elements remain on the viewport even when stacked elements have been scrolled to the top.

    false - (default) Stacked elements will scroll out of the screen once the last element to be stacked clips the top of the viewport.
